Github recon

  1. steps

    1. Check github with company name for API keys or passswords.
    2. Enumerate the employees of the company from linkedin and twitter and check their repositories on github for sensitive information.
    3. Check source code of main website and subdomains for github links in the html comments or anywhere. Search using ctl-F and search for keyword github

    1. https://github.com/BishopFox/GitGot

SQLI

  1. https://kathan19.gitbook.io/howtohunt/sqli/sql_injection
  2. Cookie injection: can only works for sqlmap above level 2. eg:sqlmap –cookie=”hint=id; id=1” -u “http://“ -p id –level=2
  3. Any other parameters needs to be forced, such as UA injection: user-agent=blabla* (–level 2)
  4. SQL injection through HTTP headers: https://resources.infosecinstitute.com/topic/sql-injection-http-headers/ (level 2 recommended as tried)
  5. Time-based blind injection: use thread=blabla to fasten it.
